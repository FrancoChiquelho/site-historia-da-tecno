Conceitos de Software

A Inteligência do Sistema

Ao contrário do Hardware, o Software é a parte imaterial (lógica) do computador. Consiste num conjunto de programas, dados e instruções que permitem ao utilizador interagir com a máquina e realizar tarefas específicas. É a ponte entre as tuas intenções e a resposta física dos componentes.

Software de Sistema

É a base indispensável. Gere o hardware e fornece uma plataforma para outros programas funcionarem.

  • Sistemas Operativos (Windows, macOS, Linux)
  • Drivers (Controladores de dispositivos)
  • Firmware (Software embebido em hardware)

Software de Aplicação

Programas desenhados para que o utilizador execute tarefas produtivas ou de lazer.

  • Processadores de Texto (Word)
  • Navegadores de Internet (Chrome)
  • Jogos e Editores de Imagem

Software Utilitário

Ferramentas focadas na manutenção, otimização e proteção do próprio sistema.

  • Antivírus e Firewalls
  • Compressores de Ficheiros (WinRAR)
  • Ferramentas de Desfragmentação

Protocolos de Licenciamento

Proprietário (Comercial) O código é privado e o uso requer o pagamento de uma licença. Exemplo: Microsoft Windows, Adobe Photoshop.
Open Source (Código Aberto) O código é público e pode ser modificado e partilhado livremente pela comunidade. Exemplo: Linux, LibreOffice.
Freeware / Shareware Software gratuito para uso, mas o código permanece fechado. O Shareware oferece um período de teste gratuito.

A Era da Nuvem (SaaS)

A tendência moderna do software é o Software as a Service (SaaS). Em vez de instalares o programa no teu disco rígido, acedes a ele através de um browser. Isto permite atualizações automáticas e acesso aos teus dados a partir de qualquer dispositivo com Internet.
