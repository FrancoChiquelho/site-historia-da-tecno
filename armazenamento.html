Matriz de Escala [UNITS]

A informação digital é quantificada através de estados binários (Bits). Para volumes de dados massivos, utilizamos prefixos de escala para definir a capacidade de armazenamento.

1 KB
1 MB
1 GB
1 TB

Módulos de Hardware [DEVICES]

Diferentes tecnologias para diferentes necessidades de velocidade e volume.

SSD_SOLID_STATE
Tecnologia baseada em memória Flash. Sem componentes mecânicos. Alta velocidade de I/O (Input/Output). Ideal para Sistemas Operativos.
HDD_MAGNETIC
Armazenamento em discos magnéticos rotativos. Velocidade inferior, mas rácio custo/capacidade superior. Ideal para backups massivos.
CLOUD_STORAGE
Arquitetura de rede distribuída. Dados armazenados em clusters remotos (Datacenters) acessíveis via protocolo TCP/IP.
TOTAL_CAPACITY_USAGE
750 GB USED 1 TB TOTAL
