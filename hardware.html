Anatomia da Máquina [01]

O Hardware constitui todos os elementos físicos e palpáveis de um sistema informático. É o "chassi" onde a inteligência do software opera. Sem uma infraestrutura física robusta, o código não tem base para execução. Cada componente abaixo é um elo vital na cadeia de processamento.

Central Unit

CPU

O processador central. Executa biliões de operações por segundo, agindo como o motor lógico de todo o sistema.

Volatile Data

RAM

Memória de acesso aleatório. Armazena os dados de curto prazo para que a CPU os aceda instantaneamente.

Mass Storage

SSD / HDD

Unidades de armazenamento persistente. Onde o sistema operativo e os teus ficheiros residem permanentemente.

Core Link

Motherboard

A placa-mãe. O sistema nervoso central que interliga todos os componentes através de barramentos de dados.

Interfaces de Entrada/Saída [02]

Os periféricos são os pontos de contacto entre o humano e a máquina. Através deles, traduzimos ações físicas em impulsos digitais e recebemos o feedback visual ou auditivo do processamento.
