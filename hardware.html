Componentes de Sistema

Anatomia da Máquina

O Hardware constitui todos os elementos físicos e palpáveis de um sistema informático. É o "corpo" do computador, sem o qual o software não teria onde ser executado. Cada componente desempenha um papel vital no processamento de dados.

Processamento

CPU

A Unidade Central de Processamento é o "cérebro". Executa todos os cálculos e coordena as instruções enviadas pelo software.

Memória Curta

RAM

Memória de acesso rápido que armazena dados temporários enquanto o computador está ligado. Crucial para o multitasking.

Armazenamento

SSD / HDD

Onde os teus ficheiros moram permanentemente. O SSD é a tecnologia moderna, muito mais rápida que os discos rígidos (HDD).

Fundação

Motherboard

A placa principal que liga todos os componentes, permitindo que a CPU, RAM e discos comuniquem entre si.

Unidades de Entrada e Saída (Periféricos)

Os periféricos são a nossa interface de comunicação com o hardware interno. Permitem-nos enviar comandos e receber resultados visuais ou sonoros.
