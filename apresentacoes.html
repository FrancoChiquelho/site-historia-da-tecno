/// TÉCNICAS DE APRESENTAÇÃO & COMUNICAÇÃO

Protocolos Visuais [GRID_LAYOUT]

Uma apresentação eficaz é a fusão entre narrativa e design. Mais do que suportes de texto, os diapositivos são interfaces visuais que devem guiar a atenção do utilizador para o objetivo central.

CONCEITO ESPECIFICAÇÃO TÉCNICA
[SLIDE_MASTER] A matriz de design global que define fontes e cores.
[LAYOUT] A arquitetura de elementos dentro do diapositivo.
[TRANSITION] Processamento visual entre diferentes estados de dados.
[ASSETS] Elementos multimédia (vídeos, HD Images) de suporte.

Diretrizes de Design [RULES]

Implementa estes padrões para garantir a máxima retenção de informação.

ESTRUTURA_LÓGICA
Introdução, Desenvolvimento e Conclusão. Uma narrativa linear e clara.
ALTO_CONTRASTE
Cores escuras sobre claras ou vice-versa para máxima legibilidade.
ELEMENTOS_MULTIMÉDIA
Utilização de ícones e vídeos para quebrar a densidade do texto.
ANIMAÇÃO_SUBTIL
Movimentos funcionais para guiar o olhar, nunca para distrair.

Fluxo de Atenção [ATTN_RENDER]

A atenção do público é um recurso limitado. Otimiza os teus gráficos para leitura instantânea.

⚠

CRITICAL: DATA_OVERLOAD

Detetado excesso de texto no diapositivo. Aplica a Regra dos 6x6 para evitar fadiga cognitiva do espetador.
