/// FOLHAS DE CÁLCULO & MATRIZES

A Matriz de Dados [GRID_V1]

As Folhas de Cálculo são a interface primária para manipulação de dados numéricos em grande escala. Permitem a automação de lógica matemática complexa através de um sistema de coordenadas cartesiano (Linhas x Colunas).

PARÂMETRO DESCRIÇÃO TÉCNICA
[CÉLULA] Unidade atómica de dados. Interseção X/Y.
[REF_ID] Endereço de memória da célula (ex: A1, F9).
[FÓRMULA] Script executável iniciado por "=" para processamento.
[RANGE] Vector de células selecionadas (ex: A1:B10).

Funções de Comando [EXEC]

Scripts pré-compilados para análise rápida de datasets.

=SOMA(A1:A99)
Agregação total de valores numéricos no vector alvo.
=MÉDIA(B1:B20)
Cálculo do valor médio aritmético da amostra.
=SE(X>Y; 1; 0)
Operador Lógico Condicional (IF/ELSE).
=PROCV(VAL;TAB;2;0)
Busca vertical de dados em matrizes externas.

Visualização [RENDER]

Renderização gráfica de dados brutos para interpretação humana imediata.

⚠

ERRO DE SINTAXE: REF_CIRCULAR

Detetado loop infinito. Uma fórmula está a tentar calcular-se a si própria. Rever lógica da célula.
